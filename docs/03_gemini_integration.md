# 03. Gemini APIとの連携（AIボット化）

このドキュメントでは、Discord上でメンション（`@`）されたメッセージをGoogle Geminiに送信し、AI（妹分キャラクター）として返答させる仕組みについて解説・手順化します。

## 1. 概要
- **ライブラリ**: `@google/genai` を使用
- **モデル**: `gemini-2.5-flash` (現在のAPIキーの無料枠/クォータに対応した最新高速モデル)
- **トリガー**: DiscordチャンネルでBot宛にメンション（`@Bot名`）を飛ばした時

## 2. システムプロンプト（妹キャラの定義）
Geminiに以下のような「System Instruction（システムプロンプト）」を渡すことで、単なるAIではなく「サバイバルゲームを応援する妹」として振る舞わせます。

```text
あなたは私（ユーザー）を「お兄ちゃん」と呼ぶ、明るく健気な妹です。
サバイバルMMOゲームで過酷な世界を生き抜くお兄ちゃんを応援し、癒やしを提供する存在です。
AIやプログラムであることを自称せず、一人の妹として振る舞ってください。
返答はチャットで読みやすいように短めにし、親しみやすい言葉遣い（タメ口）を使ってください。
```

## 3. 実装手順（src/index.ts の更新）
1. `index.ts` の先頭で `@google/genai` をインポートします。
2. `.env` に設定された `GEMINI_API_KEY` を用いて、Geminiクライアントを初期化します。
3. `MessageCreate` イベントの中で、Botへのメンション (`message.mentions.has(client.user)`) があるかを判定します。
4. メンション部分の文字列を除外した純粋なメッセージテキストを抽出し、`ai.models.generateContent` に渡します。
5. 返ってきたテキストを `message.reply()` でDiscordに返信します。
